# 🏡 Luxury Estates

A premium real estate platform designed for luxury property agencies, featuring cinematic animations, advanced property search, and a full admin management system.

---

## 🌐 Overview

Luxury Estates is a **demo platform inspired by real client demand**, built to showcase how high-end real estate brands can present properties with elegance while managing listings efficiently.

The platform combines **immersive storytelling** with **powerful backend management tools**.

---

## 💡 Problem

Luxury real estate agencies often face:

- Generic, uninspiring websites that don’t reflect brand value  
- Difficulty managing listings and media assets  
- Fragmented inquiry handling across platforms  
- Poor user experience for high-value clients  

---

## ✅ Solution

This platform delivers:

- A visually rich, cinematic browsing experience  
- A robust admin dashboard for managing properties  
- Seamless media handling with cloud storage  
- Advanced filtering for better property discovery  

---

## ✨ Key Features

- 🎬 GSAP-powered cinematic scroll animations  
- 🏡 Advanced property search and filtering  
- 🛠️ Full admin dashboard with CRUD operations  
- ☁️ Cloudinary-powered media management  
- 🔐 Authentication system for admin access  
- 🌙 Dark-themed luxury UI with gold accents  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

- **Framework:** Next.js  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** GSAP, Framer Motion  
- **Authentication:** NextAuth.js  
- **Database:** PostgreSQL (Neon)  
- **Media Storage:** Cloudinary  
- **Icons:** Lucide React  
- **Deployment:** Vercel
